Giới thiệu

Tôi là Nguyễn Đăng Lập (Lapp Security), tập trung vào lĩnh vực pentest , khai thác lỗ hổng ứng dụng và offensive security. Đang nâng trình ở cấp độ mid–high chuyên về phân tích và khai thác lỗ hổng thực tế.

Kỹ năng chuyên môn

  • Web Exploitation (Mid–High): SQLi, XSS, CSRF, IDOR, SSRF, LFI/RFI, Auth Bypass
  • Web Pentesting: Recon, Fuzzing, Subdomain Enumeration, Vulnerability Exploitation
  • OSINT (Basic–Mid): Footprinting, Domain Intelligence, Surface Mapping
  • Tooling: Burp Suite, Nmap, FFUF, Dirsearch, Subfinder, HTTP Toolkit

Tools tự phát triển

  • Scanner cơ bản (Python / JavaScript)
  • Script automation hỗ trợ recon / fuzzing
  • Một số công cụ phục vụ pentest nội bộ
