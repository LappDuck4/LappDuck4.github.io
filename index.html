Giới thiệu

Tôi là Nguyễn Đăng Lập (Lapp), tập trung vào mảng pentest web và khai thác lỗ hổng thực tế. Đang phát triển kỹ năng lên trình độ mid–high trong offensive security và phân tích bảo mật.

Kỹ năng chuyên môn

  • Web Exploitation (Mid–High): SQLi, XSS, CSRF, IDOR, SSRF, LFI/RFI, Auth Bypass
  • Web Pentesting: Recon, Fuzzing, Subdomain Enumeration, Vulnerability Assessment
  • OSINT (Basic–Mid): Footprinting, Domain Intelligence, Surface Mapping
  • Công cụ: Burp Suite, Nmap, FFUF, Dirsearch, Subfinder

Tools tự phát triển

  • Scanner đơn giản (Python / JavaScript)
  • Script automation hỗ trợ recon & fuzzing
  • Công cụ phục vụ pentest nội bộ
